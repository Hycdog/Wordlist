# iFindWord
背单词py软件

<p>1、安装python3.6
从python.org下载最新的python，并配置好环境变量(安装时点add python3.6 to PATH)</p>

<p>2、运行<b>背单词launcher.bat</b>(初次运行可能需要安装文件，建议<b>背单词launcher_admin.bat</b>)</p>

<p> 基于Windows平台开发，Linux用户可使用精简版ClipboardForLinux.</p>
